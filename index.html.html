⚙️ Configuração inicial
Cria um banco de dados gratuito no Google Firebase (~5 minutos)
  1. 1Acesse console.firebase.google.com com sua conta Google
  2. 2Clique em "Criar projeto" → dê um nome qualquer → confirme
  3. 3No menu lateral clique em Realtime Database → "Criar banco" → "Modo de teste"
  4. 4Copie a URL que aparece (começa com https:// e termina com .firebaseio.com)
  5. 5Cole abaixo e defina uma senha para o painel da mentora
